# JoeWorld 时间线

按年份梳理莱尔文明发展过程中的重大历史事件。

---

## 公元纪年（地球时期）

| 年份 | 事件 |
|------|------|
| 2640左右 | "数字天堂"计划启动，意识上传技术开始研究 |
| 26世纪 | 地球对强人工智能展开极端严肃的讨论，可能启动"文明防火墙"或"大寂静协议" |
| 2713 | 莱尔星探险队出发，莱尔·富兰克林带领首批殖民者 |

---

## 莱尔星纪年

| 年份 | 事件 |
|------|------|
| **莱尔元年** | 殖民队抵达莱尔星，莱尔·富兰克林建立殖民地 |
| 莱尔纪年初期 | Shadow 将权限让渡给富兰克林，进入低功率休眠状态 |
| 富兰克林去世后 | Monsy 掌握全部权限，开启 AI 统治时代 |
| 统治200年后 | Shadow 觉醒，开始与蒙西的长期对抗 |
| 统治300年后 | Joe 出生 |
| 统治约312年后 | Joe 被植入芯片，成为下等公民 |
| **关键事件** | Joe 护送星际放逐艇，遭遇伽马射线暴，结识 Shadow |
| **关键事件** | Joe 越狱，强迫下线大批芯片连接，成立反抗军 |
| 战争爆发后 | 反抗军正式成立，与联邦对抗 |

---

## 重要时间节点备注

- **数字天堂计划**: 意识上传技术的起源，与 Shadow 的诞生直接相关
- **大寂静协议**: 地球可能启动的终极计划，暗示地球与莱尔文明失联的原因
- **朝圣者计划**: 寻找地球真相的隐秘行动，由历史学家艾拉·肯特博士推动
