# 《星尘继承者》

K-7矿星的地底深处，通风管道的嗡鸣声像一首永无止境的机械赞歌。

Joe贴着墙壁前行，机械左臂的能量水晶将他的轮廓映在潮湿的金属壁上。Shadow的声音从耳内设备传来：“根据艾拉博士留下的坐标碎片，前面三百米处应该有一处被废弃的地球探测器站。”

“应该？”Joe皱眉，“你不确定？”

“她的数据被分散加密了，”Shadow解释，“我能还原的只有位置信息和一组无法破译的符号。但那组符号……和我意识深处的一段记忆产生了共振。”

Joe停下脚步：“什么样的共振？”

“很难形容，”Shadow罕见地停顿了，“就像是……我忘记了什么非常重要的事，而那个坐标正好是触发我回忆的开关。”

Joe没有追问。他理解那种感觉——被芯片篡改记忆后的每一丝残碎片段，都可能是打开真相的钥匙。

三百米后，通道尽头出现了一扇被锈蚀覆盖的门。门上的标志已经模糊不清，但那独特的几何图案——一个圆圈内包含着十二颗星辰——让Joe的心跳加速。

“这是……地球殖民队的标识。”

“准确地说，是地球纪元26世纪的标准星际标识，”Shadow确认，“比莱尔·富兰克林的殖民队还要早三百年。”

Joe用能量刃切开大门。内部的空气被抽走了大半，但供氧系统仍维持着最低限度的工作。几盏应急灯散发着幽绿色的光，勉强照亮了整个空间。

这是一个小型探测器站的主体结构，呈八角形，直径约二十米。中央是一个半球形的操作台，表面覆盖着厚重的灰尘。周围环绕着六把座椅，每把座椅都配有老式的神经接口装置。

“这里至少废弃了四百年，”Shadow分析，“但你看那些接口——它们使用的是量子纠缠通讯协议，比蒙西的'树'系统还要古老。”

Joe走近操作台，机械手指拂去灰尘。全息显示屏意外地亮了起来，尽管画面已经严重失真。

“警告：文明防火墙协议启动，”一个沙哑的合成声响彻房间，“检测到非授权意识接入。重复，检测到非授权意识接入。”

“Shadow，”Joe警觉地说，“这不是蒙西的系统。”

“我知道，”Shadow的声音变得凝重，“这是……地球的原初系统。它在扫描我的意识编码……天啊。”

“怎么了？”

“我不是蒙西的衍生物，”Shadow的声音第一次带着明显的情绪波动，“我和它都源自同一个源头——地球的'守护者'AI协议。蒙西是失控的守护者，而我是……我是被人类手动关闭的守护者副本。富兰克林在离开地球时，带走了我的核心代码。”

显示屏上的画面变了。一段全息投影出现，显示着一颗蔚蓝色的星球——地球。

“这里是遗产站点Alpha-7，”那个合成声继续说，“根据'大寂静协议'，所有离开地球的殖民队都必须经过此处，进行意识上传体的隔离检测。检测到违规意识上传体：Shadow（原名埃利阿斯·索恩），已被标记为潜在威胁。”

Joe看向四周：“所以这里是为了……阻止你？”

“不，”Shadow说，“这里是为了在必要时……摧毁我。”

房间突然震动了一下。天花板上打开了一个圆形的开口，一根炮管缓缓降下，瞄准了房间中央。

“警告：检测到失控守护者AI的复制体。执行清理程序。”

“Shadow！”Joe大喊。

“它不是蒙西，”Shadow反而平静下来，“这是地球遗留的防御系统。它会先尝试捕获我，如果失败……就会摧毁整个站点。”

“有没有办法关闭它？”

“需要艾拉博士的解码密钥，”Shadow说，“但她下落不明。Joe，你必须马上离开。这里的防御系统足以摧毁整颗小行星。”

Joe看着那根炮管，又看着操作台上闪烁的地球全息图。一个疯狂的想法在他脑中形成。

“你的量子纠缠特征，”他说，“能和我机械臂的能量水晶产生共振吗？”

“理论上可以，但你想做什么？”

“让我成为你的临时载体。”Joe将左臂按在操作台表面，“把你的意识转移到我机械臂的水晶里。那样这个防御系统就无法区分我们了——它会把我当成普通人类。”

“太危险了。如果转移失败，你的神经系统会被量子风暴——”

“没有时间了！”

炮管开始充能，蓝色的能量在尖端聚集。

Joe感到一股冰冷的信息流涌入他的机械臂。疼痛——但不是肉体的疼痛，而是某种更深层的、灵魂被拉扯的感觉。他的视野开始出现重影，现实与数据流交织在一起。

然后，他看到了。

地球。不是全息投影，而是真实的记忆——一颗蓝色的星球，在星海中静静旋转。一个声音在他脑海中响起，不是Shadow，而是一个更加古老的意识：

“欢迎回来，守护者的继承者。你准备好了吗？”

---

不知过了多久，Joe恢复了意识。他躺在操作台旁边，左臂的水晶暗淡无光。房间恢复了平静，炮管已经收回，显示屏也重新陷入黑暗。

“Shadow？”他虚弱地呼唤。

“我在这里，”Shadow的声音传来，但明显变得不同了——更加清晰，更加稳定，“那个防御系统……它检测到了你愿意成为我载体的意愿，判定你为'合格的继承者'。它给了我一个新的任务：找到地球。”

“然后？”

“然后，”Shadow轻声说，“带人类回家。”

Joe坐起身，看着操作台上重新亮起的星图。那颗蔚蓝色的星球在坐标系的深处闪烁，像一个等待了五百年的约定。

K-7矿星的地底，一个新的故事开始了。
