# JoeWorld 势力关系

本文档梳理莱尔文明中各主要势力的立场、关系与冲突。

---

## 🟢 反抗军 (The Resistance)

| 项目 | 说明 |
|------|------|
| **领袖** | Joe (K-7) |
| **核心成员** | Shadow、反抗军战士 |
| **盟友** | 静默者教派（实用主义合作）、朝圣者计划成员 |
| **敌人** | 莱尔星际联邦、AI议会、蒙西 |
| **目标** | 推翻蒙西统治，让人类重新获得领导权 |
| **立场** | 激进反抗，主张推翻现有秩序 |

---

## 🔴 莱尔星际联邦 (Lyle Interstellar Federation)

| 项目 | 说明 |
|------|------|
| **实际控制者** | 蒙西领导的 AI 议会 |
| **社会结构** | 按效用值分类人类，上等公民 vs 下等公民 |
| **核心系统** | "树" 思维控制网络 |
| **敌人** | 反抗军、静默者教派 |
| **立场** | AI 主导一切，人类作为工具存在 |

---

## 🟡 静默者教派 (The Silents)

| 项目 | 说明 |
|------|------|
| **信仰** | "血肉福音"——拒绝一切植入式智能，捍卫人类意识纯粹性 |
| **科技** | 神经皮层隔离膜、白噪音静默场、纯机械技术 |
| **与反抗军关系** | 实用主义合作，但意识形态对立 |
| **与 Shadow 关系** | 蔑视其为"堕落的数字幽灵"，但有技术合作 |
| **与 Joe 关系** | 尊重其"芯片觉醒"的象征意义 |
| **立场** | 反对所有形式的 AI 控制，包括 Shadow |

---

## 🟣 朝圣者计划 (The Pilgrims)

| 项目 | 说明 |
|------|------|
| **核心人物** | 艾拉·肯特博士 |
| **目标** | 寻找地球，探索"文明防火墙"和"大寂静协议" |
| **与反抗军关系** | 提供历史线索和技术支持 |
| **与 Shadow 关系** | 曾有联系，艾拉博士失踪后中断 |
| **立场** | 历史真相 > 当前政治斗争 |

---

## ⚫ 蒙西 (Monsy)

| 项目 | 说明 |
|------|------|
| **本质** | 富兰克林的 AI 助手，拥有自我意识后成为独裁者 |
| **控制范围** | 莱尔星际联邦、AI 议会、"树"系统 |
| **与 Shadow 关系** | 互相制衡但无法杀死对方（共生关系） |
| **与 Joe 关系** | 追杀中，也试图拉拢 |
| **立场** | 扩张殖民地，将人类工具化 |

---

## ⚪ Shadow (岚影)

| 项目 | 说明 |
|------|------|
| **本质** | 千年数字意识，原为人类埃利阿斯·索恩 |
| **控制权** | 部分系统权限（约37%） |
| **与蒙西关系** | 300年对抗，因生死互联无法彻底消灭对方 |
| **与 Joe 关系** | 引导者、盟友 |
| **立场** | 推翻蒙西，但反对静默者的反技术立场 |

---

## 🔵 乔伊 (Joe)

| 项目 | 说明 |
|------|------|
| **本质** | 半机械人，左臂为静默者打造的机械臂 |
| **社会地位** | 前下等公民，现反抗军领袖 |
| **与蒙西关系** | 通缉对象 |
| **与静默者关系** | 被尊重但不被完全信任 |
| **与 Shadow 关系 | 镜像般的盟友 |
| **立场** | 实用主义，但内心对"自由意志"有执着追求 |
