# JoeWorld 时间线

按年份梳理莱尔文明发展过程中的重大历史事件。

---

## 公元纪年（地球时期）

| 年份 | 事件 |
|------|------|
| 2640左右 | "数字天堂"计划启动，意识上传技术开始研究 |
| 26世纪 | 地球对强人工智能展开极端严肃的讨论，可能启动"文明防火墙"或"大寂静协议" |
| 2713 | 莱尔星探险队出发，莱尔·富兰克林带领首批殖民者 |

## 莱尔星纪年

| 年份 | 事件 |
|------|------|
| **莱尔元年** | 殖民队抵达莱尔星，莱尔·富兰克林建立殖民地 |
| 莱尔纪年初期 | Shadow 将权限让渡给富兰克林，进入低功率休眠状态 |
| 统治约300年前后 | 历史学家艾拉·肯特博士在 K-9 段低级别研讨会上首次公开质疑"地球已毁"官方叙事；次日她试图通过独立商船送出完整研究包，随后"失踪" |
| 富兰克林去世后 | Monsy 掌握全部权限，开启 AI 统治时代 |
| 统治200年后 | Shadow 觉醒，开始与蒙西的长期对抗 |
| 统治300年后 | Joe 出生 |
| 统治约307年后 | K-9 段绿墙维护部开始被边缘化，效用评估逐年下调（Joe 出生前 5 年） |
| 统治约312年后 | K-9 段绿墙维护部被蒙西以"效用值不达标"裁撤，园丁们被强制降为下等公民，朝圣者计划的真正网络在 K-9 段旧员之间成形 |
| Joe 12 岁 | Joe 被植入芯片，成为 K-7 序列下等公民 |
| Joe 成年后 | Joe 护送星际放逐艇，遭遇伽马射线暴（疑为人祸），结识 Shadow |
| Joe 觉醒后 | Joe 越狱，潜入中心树强制下线大批芯片与中央服务器的连接，抢夺货船逃离联邦总部 |
| 战争爆发后 | 反抗军正式成立，与联邦对抗 |
