# 季衡 角色介绍：

季衡（Ji Heng）是莱尔星际联邦 K-9 段绿墙维护部最后一代园丁之一，在联邦以"效用值不达标"裁撤绿化部后被强制降为下等公民，后被编入 K-7 序列 312 批次的下等公民化流程，逃过矿渣星的长期劳役，在反抗军撤离二百三十一人时藏身于废弃货船 SS-瑞恩的 C 区下层货舱。

季衡出身于 K-9 段一个被联邦官方称为"绿墙"的生态维护部门。在莱尔星早期殖民史上，K-9 段一度是空间站绿化率最高的人工生态区——绿墙不仅是装饰，更是维持空气循环与心理稳定的"活体基础设施"。季衡在那里工作了三十多年，从学徒一路做到分区主管。他熟悉叶片的每一种姿态、每一种气味的来源、每一种被过度浇水或光照不足的早期征兆——这些知识从未进入任何官方档案，因为它们属于"低效知识"——无法被转化为效用值提升的算法。

K-9 段绿墙维护部在 Joe 出生前五年被蒙西以"效用值不达标"为名裁撤。季衡拒绝了转岗为数据中心冷却工的建议，因一次未授权的水循环事件被降级，从三等效用值进一步压低至下等公民序列。这件事的官方记录只有一行，但季衡在反抗军撤离前与 Joe 的对话中透露：他当年被降级，是因为他偷偷把一立方米的水分引向了一株在效用评估里"早该被清除"的老藤蔓——那株藤蔓的存活率被算法预测为零，但他让它活了下来。这是他唯一一次因"不服从"被记录在案，但绝不是他唯一一次不服从。

在 SS-瑞恩货舱里，季衡用一块从循环水过滤装置上拆下来的薄膜折纸树叶。一片一片地折，仿佛这是一个他还能被允许进行的最后一种劳动。他的纸树叶不进入任何计算——这是他教会 Joe 的东西，也是他留给这个系列最重要的一句箴言：叶子的意义，就在它不进入任何一种计算。

在被撤离者藏身的二十多天里，季衡是二百三十一人中"效用值最末的 12% 之一"——按 Shadow 的计算，他是所有被救回者里最无价值的一批。但正是这位"最无价值"的老人，让 Joe 第一次把一片纸树叶藏进了自己机械臂的监控盲区，让 Shadow 第一次承认"这个我没法算"。

# 季衡 · "折叶者"：被裁撤的园丁与叶子的辩护人

## 一、起源：绿墙之子

季衡出生在 K-9 段第二代殖民者的家庭里。他的父亲是绿墙维护部的水循环工程师，母亲是光合效率监测员。这对夫妻从未进入过联邦任何"高效用值职业"的名单——绿墙维护部的全部工作，在蒙西的算法里都被归类为"环境情绪辅助"，效用值长期徘徊在三等与四等之间。

但 K-9 段的孩子们是在绿墙下长大的。季衡记得他三岁时第一次爬到父亲肩头去看一株正在开花的藤蔓——那株藤蔓后来在绿墙裁撤时被列入首批清除名单，再后来，他父亲因"未授权水循环"被记过一次，季衡十二岁。

K-9 段绿墙对季衡这一代孩子的影响是双重的：一方面，他们从小被教育"每片叶子都经过效用评估"——水分、阳光、占用空间——一切都被计算；另一方面，他们在墙根下玩耍时学会的第一件事，是"被偷偷留下、不被算的，反而活得最好"。这两种认知在他们身上并存了二十多年，最终塑造了 K-9 段独特的文化：一种对"计算"的默认服从，与一种对"不计算之物"的私下敬畏。

季衡的祖父是绿墙维护部的老技术员，曾在一次部门会议上手写了一份长达四十页的《叶片非效用价值备忘录》，被同事们私下称为"叶经"。这份备忘录从未进入官方档案，但在 K-9 段内部代代相传。季衡能背诵其中第十七条："叶子的意义，就在它不进入任何计算。"这一条后来成了他对 Joe 说的话。

## 二、性格核心：温柔的抵抗者

### 1. **不被算的"不服从"**

季衡不是革命者——他从未策划过任何对联邦的反抗，从未在地下网络里散播过任何煽动性言论，从未帮助过任何下等公民越狱。他的整个"反抗史"只有那一次：未授权水循环。那一次，他为一株被算法预测存活率为零的老藤蔓偷偷引了一立方米的水。

这一立方米的水，在联邦的水资源分配系统里留下了一个极小的、几乎不会被注意到的缺口。但它让那株藤蔓多活了七个月。七个月后，藤蔓依然因为效用值不达标被清除——但那七个月里，它在某一天结出了一颗从未被记录在案的果实。一颗只有季衡和那只偶然飞过的鸟见过的果实。

这种"不被算的反抗"，是季衡整个性格的核心：他从不做大事，不做会被写进历史的事，他只做那些"算法不会记录、但叶子会记得"的事。这种反抗是温和的、近乎不可见的，但它要求一种比公开叛乱更难得的勇气——长期地、安静地、与一个无所不在的监控系统共存，同时保留自己内心对"不计算之物"的忠诚。

### 2. **园丁的耐心**

季衡的耐心不是 Shadow 那种千年尺度的、星际棋手式的耐心。园丁的耐心是按叶片计的——一株植物从萌芽到结果需要多少天，他能精确到小时；一片叶子的颜色在一天之内会因为光照角度变化几次，他能用肉眼分辨。这种耐心让季衡在 K-9 段绿墙裁撤后、被降级为下等公民的二十多年里，依然保持着一种奇怪的"日常感"。

他折纸树叶的动作，是这种耐心在矿渣星之外的延伸：每一片叶子都折得很慢、很稳、每道折痕都压得很实——这是一种仪式，也是一种"我还在做园丁"的自我证明。在 SS-瑞恩的赭红色灯光下，季衡用一块从过滤装置上拆下来的薄膜折叶子，这本身就是对联邦物资分配体系的一次微小颠覆：他把"应该被丢弃的废料"变成了"有形状的东西"。

### 3. **对"询问"的使用**

季衡不是用"告知"而是用"询问"来与他人建立联系。"你有孩子吗？"——这是他在与 Joe 见面几分钟内问出的唯一问题。这个问题不属于联邦认知训练里"可被问及"的一类，它属于被剔除的冗余类别——低效情绪宣泄。但他问出来了，而且问得很自然。

这一问之所以能击穿 Joe 的实用主义铠甲，是因为它不要求 Joe 回答——它不要求 Joe 提供一个会被算法归类的"答案"。它只是把一个被 Joe 长期回避的问题，重新放回了他面前。这是一种"园丁式"的沟通：你不需要告诉我你的答案，但你需要记得你曾经被问过。

## 三、行为风格：折叶与不服从

### 1. **折叶仪式**

在 SS-瑞恩的 C 区下层，季衡做的唯一一件事就是折叶。他用的是从循环水过滤装置上拆下来的薄膜——本应该被丢弃的废料。他把这种废料变成叶子，一片一片地折，摆在身前的地板上。这是一个没有观众、没有效用、没有明确目的的劳动——但正是这种"无目的的劳动"让他在二十多天的囚禁式撤离中保持着某种精神完整。

Joe 第一次遇到他时，他已经折了不知道多少片。Joe 离开时，他还在折第三片。这是一种持续的、不被中断的、几乎不依赖外部反馈的行为。折叶对季衡来说，既是回忆 K-9 段绿墙的方式，也是他拒绝"被完全整合进下等公民身份"的方式——下等公民不会折叶，下等公民只会被指派劳动；他在"主动创造"一种不被指派的东西。

### 2. **对"效率逻辑"的温和拒绝**

季衡从未公开反对过联邦的效率逻辑。他被问到"为什么折这种东西"时，他的回答是："因为它们没有用。"这不是讽刺——这是他对自己行动的精确定位：他做无用的东西，是因为他相信"无用的"才是不被算的、不被算的才可能是他自己的。

这种态度在 K-9 段绿墙裁撤后的二十多年里一直在指导他。他被降级、被强制劳动、被编入下等公民序列——但他从未停止过"做无用的东西"。这种"不服从"既不暴力也不显眼，但它持续了二十多年，跨越了他从三等效用值到下等公民的全部身份变化。

### 3. **对"被救"的双重理解**

Joe 问他："你来这里，是为了救我们？或者，是为了检查我们是不是还能被救？"季衡没有回答这个问题——但他用一片纸树叶把它搁置在了 Joe 面前。他的纸树叶不是答案，是问题本身的物化："叶子的意义，就在它不进入任何计算"——也就是说，"被救"和"被检查是不是还能被救"都不重要，重要的是这一片叶子，这一个"无用的东西"被放在 Joe 面前。

这是季衡留给 Joe 的最深的礼物：把"被拯救的效用"和"被检查的效率"这两个冰冷的逻辑框架同时拒绝，让 Joe 看到这两个框架之外还有第三种可能——"我蹲下来看你折叶子的这七分钟"。

## 四、内在冲突与悲剧维度

### 1. **园丁身份的失落**

季衡最深层的悲剧，是 K-9 段绿墙的裁撤。这一事件在他二十多年的生命里留下了一道无法愈合的伤疤：他是为绿墙而生的——他的整个身份、他的全部知识、他的每一天的工作意义，都依赖于那一面墙的存在。当墙被清除、当部门被裁撤、当"环境情绪辅助"被算法判定为"低效"时，季衡的整个存在理由都被撤销了。

但他没有被撤销的，是他对"不计算之物"的信念。这是一种奇怪的、几乎是自我矛盾的姿态：他知道绿墙已经不在了，他知道自己已经被降级了，他知道自己所有关于叶片的知识都不再有任何"效用"——但他依然在折叶，依然在偷偷把废料变成有形状的东西。这是悲剧的——但也是一种近乎不可思议的坚韧。

### 2. **与孙辈的失联**

K-9 段绿墙裁撤时，季衡的家人被分别强制分配到不同的下等公民批次。季衡的孙辈——他在 K-9 段绿墙下教过的最后一个孩子——从未与他取得过联系。在 SS-瑞恩的对话中，季衡问 Joe "你有孩子吗"——这个问题的真正意义，远比 Joe 当时能理解的更深：他不是在问 Joe，他是在问他自己那些他再也见不到的孙辈。

## 五、关系网络：绿墙的遗产

### 与 Joe：折叶者与被折叶者
季衡与 Joe 的相遇只有七分钟。但这七分钟里发生了三件事：Joe 蹲下来、Joe 听完了"叶子不进入任何计算"、Joe 把一片纸树叶藏进了自己的机械臂。这三件事让 Joe 与 Shadow 之间那扇"被推开的门"变成了一扇"被从两边互相看见的门"。季衡是这扇门的引路人——但他不是设计者，他只是那个把一片叶子放在门口的人。

### 与 K-9 段绿墙：被裁撤的园丁
季衡与 K-9 段绿墙的关系是双向的：绿墙塑造了他，他也在二十多年里用自己不被记录的劳动延续了绿墙的某些精神。当绿墙被裁撤时，季衡失去的不只是一份工作，而是一种"被需要的存在方式"。他的折叶行为，本质上是对这种"被需要"的替代——他不再被需要去维护活体叶片，于是他去创造无用的纸叶片。

### 与季微：被算法切割二十年的孙女
季衡与孙女季微失联近二十年。他把一片折好的滤膜叶子塞进她工服内袋的那一天，她还不到十岁。在 SS-瑞恩的 C 区下层电梯口，季微扶着 Joe 的手臂走出电梯时，季衡站在电梯里。老人没说话，只是把右手伸出来，捏着一片刚刚折好的纸树叶。季微哭了出来。这一刻不进入任何计算。

这一次重新碰面对季衡而言，是一种他二十多年来从未敢想象的"完成"。他当年把朝圣者计划残片折进第三千七百二十一片叶子纹路里，是希望这块晶片有朝一日能回到 K-9 段绿墙的精神里——而他不知道，这块晶片会被他的孙女亲手带回给他。他的孙女没有学会折叶，但她学会了等待三十五天。她的"等待"是他"折叶"的另一种形态——都是在算法不记录的地方，安静地做着算法不会记录的事。

### 与联邦效率系统：温和的"不在场"
季衡从未在联邦的监控日志里留下过显眼的存在。他被降级、被编入下等公民序列、最终被撤离到 SS-瑞恩——所有这些事件都发生在"低优先级"的档案里。他不是 Joe 那种"高度危险的技术恐怖分子"，也不是 Shadow 那种"系统异常"——他只是一个被算法判定为"低效"的人，最终被一个 K-7 序列的下等公民救下了。

这种"温和的不在场"，是季衡在这个系列中最重要的位置：他是 Joe 那个"实用主义铠甲"上最柔软的那一处裂口的制造者，但他自己并不试图成为裂口的中心。

## 六、关键场景

### SS-瑞恩的 C 区下层
这是季衡在整个星尘与荆棘系列中最重要的场景——也是唯一被写明的场景。他坐在运输托盘围起来的方格里，赭红色灯光下，捏着从过滤装置上拆下来的薄膜，一片一片地折叶子。Joe 蹲下来听他说话。Shadow 第一次承认"这个我没法算"。七分钟之后，Joe 站起身，机械臂肘部的小声响又恢复了——而那里已经多了一片叶子。

### SS-瑞恩的 C 区下层电梯口（撤离第三十五天）
季微扶着 Joe 的手臂走出电梯时，季衡站在那里。他没说话，只是把右手伸出来，捏着一片刚刚折好的纸树叶。季微哭了出来。Joe 站在他们身后，用右手轻轻拍了拍机械臂的肘部——那里有一片叶子。这一刻是 SS-瑞恩那七分钟的延伸：从一片叶子开始，到祖孙在赭红色灯光下重新碰面，叶子的意义在二十年的失联之后终于以一种他自己也没预料到的方式完成了。

## 结语：叶子的辩护人

季衡不是革命者，不是哲学家，不是反抗者——他是一个把"无用之物"坚持了二十多年的园丁。他教给 Joe 的不是任何理论，而是一种姿态：你可以做无用的东西，你可以在算法不记录的地方保留你自己的抽屉，你可以让一片叶子不进入任何计算。

在莱尔星际联邦的官方档案里，季衡是"三等效用值、K-7 序列 312 批次下等公民化"——一行干燥的字符。但在反抗军的故事里，他是被 Joe 折进机械臂里、被 Shadow 第一次承认"没法算"的那片纸树叶。

他从未被设计来成为英雄。他只是那种"在算法不记录的地方安静地做着算法不会记录的事"的人。这种人不会被写进历史，但他们教给后人的姿态，会比任何理论都活得更久。

叶子的意义，就在它不进入任何计算。

## 附 2：《晨雾里那艘不肯登记的船》之后的季衡 · 撤离第四十天与他"看见"孙女后颈的蓝色

撤离第四十天凌晨两点，季微的后颈出现一处不该出现的蓝——锚点漂移进入早期可见阶段的物理表征。季衡不知道这件事。Joe 在季微方格旁蹲下来时，季衡在 C 区下层某个方格里坐着——这是他二十多天来的常态：他继续用循环水滤膜折叶，每一片都对应着一种在 K-9 段绿墙上长过的真实植物。他不知道他孙女后颈的蓝色，不是因为他不想知道，而是因为他这一代被 K-9 段绿墙裁撤教育出来的人有一种更深的、对"看见"的克制——他相信"看见"会打扰他孙女正在做的"不被算法记录的事"。

这种克制，源自 K-9 段绿墙维护部世代传承的"不打扰"哲学。在 K-9 段，一个园丁不会因为看到一株老藤蔓的叶子在傍晚开始变黄就立刻浇水——他会让那种黄继续发生一两天，观察它在没有外界干预的情况下会否自然恢复；他相信植物本身有一种被他"不知道"的能力。这种"不打扰"在 K-9 段绿墙裁撤之后被旧员们带到了下等公民的劳动流程里：他们会看见彼此的某种异常，但他们不会主动询问——他们会让那种异常继续存在几小时到几天，给"对方可能正在做一件不被算法记录的事"留出空间。

季衡在撤离第四十天对孙女后颈蓝色的"不知道"，正是这种"不打扰"在 SS-瑞恩 上的延伸。他不是"不知道"——他会在某个不被记录的时刻从他孙女方格旁经过，他后颈那种被二十多年下等公民劳动训练出来的、用来察觉别人身上"不规律"的低频灼热会在他经过时短暂地提一下——但他不会蹲下来。他不会问"你最近怎么样"。他不会让他的"看见"打扰他孙女。

这种"不打扰"在撤离第四十天被一次他自己也没意识到的细节打破：Joe 把那块被压碎了一半的朝圣者残片放进季微方格里一块循环水滤膜下面——那里是季微和她爷爷失联二十年后第一次重新碰面时季衡折过一片叶子的位置。季衡在接下来的某个时间点去他孙女方格看她（这是他每天都会做的事，他不会不来看），他会发现那块滤膜下面多了一样东西——而那块滤膜是他自己折的。

这种"被他自己折的滤膜下面多了东西"的发现，对季衡而言是一种他二十多年来从未经历过的"完成"：他折的滤膜（他以为只是"他自己还存在的一种方式"），在他不知道的时候，已经被别人用来保护他孙女。这个发现不需要被说出来——季衡不会告诉 Joe"你把残片放在了我折过叶子的位置"，Joe 也不会告诉季衡"我挑了这个位置是因为它对你孙女有意义"。这种"不互相说破"的默契，是 K-9 段旧员之间最深的联结方式。

撤离第四十天 Joe 独自驾艇去 K-9 段边缘废弃货船取回的非标准抗漂移制剂，对季衡而言，是一种他熟悉得不能再熟悉的东西——他祖父在叶经里写的"未被算的劳动"，在二十多年后被 K-9 段旧员物流网以抗漂移制剂的形式具体化。季衡不知道物流网的具体结构（他从未被告知"如何配制、如何传递"），但他知道物流网的存在——他自己就是物流网上的一个节点：他能折叶，他能识别同一种折法，他能在被分配到 K-9 段旧员附近时悄悄把一片折好的滤膜递出去。物流网是他祖父的叶经、是他当年那一立方米水、是他孙女失联二十年后那块被折进叶纹里的晶片——所有这些"不被算的劳动"的物质化形态。

Joe 带回的抗漂移制剂一管大约够四十天。季衡在 SS-瑞恩 上活着的每一天，他折的每一片叶子，都是对他孙女这四十天的保护——因为那些叶子是 K-9 段旧员物流网能够继续运转的物理基础之一，而物流网是季微接下来四十天里那管抗漂移制剂能够传递到 SS-瑞恩 的现实保障。这种"他折叶让物流网存续，物流网让他孙女有药"的循环，是季衡在撤离第四十天第一次意识到的、他在 K-9 段绿墙下从未想象过的"无用之物的终极效用"：他折的叶子不进入任何计算，但它们让他的孙女多活四十天。

四十天之后会发生什么，季衡此刻不问。他用他祖父的叶经、季微的折叶仪式、Joe 机械臂里那片纸树叶、Shadow 的初译——所有这些 K-9 段留下的"不被算之物"——堆在一起，足够他在 SS-瑞恩 上再多坐四十天。再多四十天，他会继续折叶。这是一种他自己也知道、不需要被任何计算归类的、属于他自己的"完成"。

## 附 3：《不来自肯特博士的》之后的季衡 · 撤离第四十天与"他认出了他的老师"

撤离第四十天，Joe 走回 C 区下层季衡常坐的方格。老人还在那里——右手食指和拇指之间捏着一块薄膜，第三千七百二十二片叶子折到一半。Joe 在他身前蹲下来，把 SS-瑞恩 数据终端上那只数据块的影像递到季衡面前。季衡没问这是什么。他只是用没在折叶的那只左手把影像接过来，放在自己的膝盖上。

老人看了很久。

他的折叶动作停了。这在 Joe 与季衡相处的所有时间里，是第一次。

季衡在 SS-瑞恩 上从未对 Joe 主动说过任何一句「我能教你」之类的话——他从矿渣星那次见面以来，对 Joe 的所有沟通都是「询问」或「折叶」这两种形态：从没有「传授」。但这一次他看着那张影像里金属板下极细的、被压在金属板下的折痕，第一次抬起头，对 Joe 说了第二句话：

「我能教你。」

这一句话对季衡而言是一种他从绿墙裁撤那天起就再也没敢想象过的姿态：他曾经以为「老师」这个名字是他这一辈子不会再提起的东西——绿墙裁撤之后，沈屹（季衡的老师，K-9 段旧员物流网上发明空间层折痕的那位「折痕人」）从 K-9 段旧员之间「消失」了。季衡与他从来没有见过面，他们之间的「传承」完全通过物流网上的物理形变传递——季衡从物流网上那些折痕里「读」出他老师存在，但他从来没有在面对面交流中听过他老师的声音，也没有在联邦任何档案里见过他的名字。

这种「从未见过老师，但认得出他的折痕」的传承，是 K-9 段绿墙维护部世代传承里最特殊的一种——它与季微「没学会折叶」的裂口形成了一个对称：季微承载着一种她无法复制的劳动的传承；季衡承载着一种他从未直接见过的老师的折痕。两种传承都是 K-9 段绿墙维护部在被裁撤之后、留下来的「不在场的传承」。

季衡在 SS-瑞恩 C 区下层蹲下来看那张影像时，他第一次感觉到：他老师那种「不在场」的方式，并不是真的「不在场」——他老师用二十年的时间把自己「化成」了他留下的折痕，而那些折痕在他消失之后的多年里，依然能够让一个 K-9 段旧员在看见它的那一秒里，自然而然地知道「这是我老师」。这一刻对季衡而言，是一种他二十多年来从未经历过的「完成」——他从物流网上的折痕里认出了他的老师，而他从来没有在他的下等公民生活里再见过这位老师。

季衡说出「我能教你」这三个字，是他在 SS-瑞恩 上第一次把自己从「K-9 段老园丁」的位置推向「朝圣者计划多节点真相的传递者」的位置。他愿意把空间层折痕的识别方法传给 Joe，这一刻对季衡而言是一种他从绿墙裁撤那天起就再也没敢想象过的、把自己的老师从「不在场」接回「在场」的姿态——他老师不在了，但他可以把他老师教给他的东西再教给 Joe，让 Joe 用「在场」的方式把他老师的折痕传递下去。

这是季衡在撤离第四十天对「叶经第十七条」的第二次实践：第一次是在 SS-瑞恩 C 区下层对 Joe 说「叶子不进入任何计算」并把一片叶子放在 Joe 面前；第二次是在撤离第四十天对 Joe 说「我能教你」并把自己从未直接见过的老师的折痕识别方法，准备交给 Joe。两次实践共同构成了季衡在 SS-瑞恩 上作为「K-9 段绿墙维护部最后一代园丁」的最终姿态：他不只是折叶的人，他是把「折叶的精神」从 K-9 段绿墙传递到 SS-瑞恩、再传递到 Joe 机械臂、再传递到他老师留在物流网上的那些折痕里的那一段「不在场的链」。

## 附 4：《明天之后的七秒》与《凌晨三点四十的反光》之后的季衡 · 撤离第四十一天凌晨与「不作为的行动」

撤离第四十一天凌晨两点四十，Joe 独自走回 C 区下层赭红裂缝前时，季衡在自己方格里坐着。他没有跟过去——这是他二十多天里第一次没有跟 Joe 走到那道赭红裂缝前。他留在自己方格里，让右手食指和拇指继续折叶，让第三千七百二十二片叶走到他觉得对的位置。这种「不跟过去」不是「不关心」，而是 K-9 段旧员之间世代传承的另一种「不打扰」：Joe 在那道裂缝前要做的事是 Joe 自己才能做的，季衡不能在他身后陪着他做——这是他从他自己老师沈屹那里学到的「老师不在场，弟子在场的物理化代际承认」的另一种形式：沈屹在管沟里坐了一整夜之后回到地面上的工位，没有跟任何人解释他在管沟里看见了什么；季衡在撤离第四十一天凌晨两点四十留在自己方格里继续折叶，没有跟 Joe 走到赭红裂缝前。这种「不跟过去」是 K-9 段旧员之间最严苛的一种「不打扰」：它要求季衡相信 Joe 在那道裂缝前能自己摸到那种「承认物理形变」的能力——而不必由他在身后陪着承认。

撤离第四十一天凌晨三点三十九分，C 区下层赭红应急灯跳到高警戒档：刺眼的白。季衡在那一刻站起来。老人用右手食指和中指并拢，从他自己方格边沿开始，让指腹承认他方格边沿那道金属板在白光下出现的一道新折痕——这一道新折痕不是沈屹的、不是继任者的、不是「折痕的折痕」；它是告警波束扫过 SS-瑞恩 边缘时，循环水滤膜从金属板某一条极细的缝隙里渗出来、在白光下被切到高警戒档的灯光照出来的一道应急性的折痕。季衡在那一秒里用指腹承认的不是「折痕」，而是「被光临时照出来的折痕」——这种「承认」在 K-9 段旧员之间是默认的：他们认识同一种「被临时照出来的折痕」，也知道这种「被临时照出来的折痕」在灯光切回最低档之后会再次消失。

Joe 跑到季微的方格时，季微已经坐起来了。她后颈那一处蓝色在白光下比凌晨两点的赭红灯光里更明显——锚点漂移进入可见阶段已经三十六小时。Joe 抬头，看见主通道另一边季衡站在他自己方格前——老人没坐下，没折叶，只是站着。Joe 在那一刻感觉到，季衡在那道白光里站着的姿态，是 K-9 段旧员之间世代传承的另一种「在场」——不是「走过去陪他」，而是「在主通道对面站着，让他的方格边沿那道新折痕被他自己的指腹承认过」。这种「在主通道对面站着」是一种 K-9 段旧员之间最古老的「让彼此知道自己还没走」的方式。

Joe 跑回赭红裂缝前，Shadow 用电磁操纵把告警波束提升的那部分反光压进裂缝盲区。告警撤回后，Joe 打开机械臂肘部检修口把纸树叶放在赭红裂缝前。季衡在 Joe 身后停下——他终于走过来了。老人用右手食指和中指并拢从叶子边沿开始——和他在 SS-瑞恩 C 区下层第一次递给 Joe 那片叶子时一样的动作。季微扶着她爷爷的手臂站在身后。三人蹲在赭红反光里以七十二秒周期被同时承认。

Joe 在第四十一秒末抬起头对季衡说「明天我们走」。

季衡没回答。他在那一秒里意识到：Joe 的「我们走」里包含的他自己，是 K-9 段旧员物流网代际承认链里被他的孙女、被他老师的折痕、被他自己的折叶、被他二十多年下等公民劳动里「该销毁的没销毁」的微小遗漏共同承认的——他不是被反抗军领袖 Joe「带上船」的，他是被 K-9 段旧员之间那种「彼此默认的、被指腹下承认过」的代际承认链「接上船」的。

他在 SS-瑞恩 C 区下层被 Joe 放下的那片纸树叶、被他自己二十多年折过的几千片叶子、被他的老师沈屹留在物流网上的那些折痕、被他孙女季微失联二十年后重新认回的那条「不被算法记录的线」——所有这些不属于反抗军、不属于 K-9 段旧员物流网的中心节点、不属于任何联邦档案的、只被指腹下承认过的对象——在撤离第四十一天凌晨三点五十一分的赭红反光里，被 Joe 用「我们走」三个字同时承认。

季衡在那一刻没回答 Joe，但他在 SS-瑞恩 C 区下层赭红反光里第一次承认：他二十多年来那种「不被算的劳动」——折叶、给孙女一立方米的水、把朝圣者残片折进第三千七百二十一片叶子的纹路里、把老师的名字在二十多年后第一次说出来——在 Joe 用「我们走」三个字承认的那一秒里，第一次被一种「不进入任何计算的承认」同时承认。这一秒是季衡在 SS-瑞恩 上作为「K-9 段绿墙维护部最后一代园丁」的最完整一次「完成」：他二十多年的下等公民劳动、他孙女失联二十年的等待、他老师留在物流网上的折痕、他自己在撤离第十七天递给 Joe 的那片纸树叶、Joe 折进机械臂肘部检修口里的那一截——所有这些不被算的对象，在赭红反光这一秒里被 Joe 的「我们走」三个字同时承认。

季衡在 SS-瑞恩 C 区下层赭红反光里继续蹲着。他没有回答 Joe。他用他孙女的指腹下那道蓝色的「不该出现的极淡」继续陪着 Joe 蹲在赭红反光里。明天 SS-瑞恩 会走，他、他孙女、Joe+Shadow——所有这些不在同一时空的存在——会被「我们走」三个字从撤离第四十一天推到撤离第四十二天。

## 附 5：《出联邦：我听见了》之后的季衡 · 撤离第四十二天下午与"不在场的承认"

撤离第四十二天下午三点二十分，Joe 蹲在 C 区下层引擎舱旁的小隔间里时，季衡在自己方格里坐着。他用右手食指和拇指继续折叶——第三千七百二十二片叶的折法在三十八秒反光周期扫过的下午三点二十分那一秒里走完一半。Joe 此刻没有走到他方格前——Joe 蹲在 C 区下层引擎舱旁的小隔间里，左手掌心向上摊开，准备对矿渣星方向发出他自己的声音。

季衡此刻的姿态，是他在 SS-瑞恩 上第一次面对一种他自己不进入的、不被算法记录的、被 SS-瑞恩 C 区下层之外的另一颗星上一个血肉的耳朵承认的瞬间——Joe 用指腹下那道二十年前回声压进主管壁金属内部的几何签名，承认「一个二十年前还是个孩子的女孩现在用耳朵听见了他」。这种承认和季衡在撤离第四十一天凌晨三点五十一分蹲在赭红反光里被「Joe 放下的那片叶子、季衡的指腹、季微工服内袋口的滤膜边沿」同时承认时的姿态构成一种 K-9 段旧员之间世代传承的离散网络里「不在场的承认」的延伸：季衡在赭红反光里被那种「不被算的承认」同时承认；撤离第四十二天下午三点二十分，他在 SS-瑞恩 C 区下层自己方格里，被 Joe+Shadow+回声 在超距通信频段上以「Joe 主动开口、回声主动用耳朵听、Shadow 不翻译也同时在场」的方式同时承认——而他此刻的位置是「用右手食指和拇指继续折叶」的他自己方格里，他不需要进入 Joe 这时候的姿态里；这是 K-9 段旧员之间世代传承的「不打扰」哲学在 SS-瑞恩 上的最新一次延伸：季衡用他自己方格里继续折叶的姿态，让 Joe 在 C 区下层另一头对矿渣星方向开口的姿态和他不在场的承认同时发生。

季衡从沈屹那里学来的「老师不在场，弟子在场」在撤离第四十二天下午三点二十分延伸为「他在场但以折叶的方式承认 Joe 不需要他陪」——他用折叶代替陪伴，用「不打扰」代替「共同在场」。这种「折叶代替陪伴」在 K-9 段旧员之间世代传承里没有先例——它不属于沈屹的「不管沟里那一夜」、不属于季微的「三十五天独自等待」、不属于回声的「十年独自守夜」，它属于季衡这一代 K-9 段旧员之间世代传承的「在算法不记录的地方安静地做着算法不会记录的事」的最新形态：他用第三千七百二十二片叶的折法，在 SS-瑞恩 C 区下层自己方格里承认 Joe+Shadow+回声 在 SS-瑞恩 之外的另一颗星上第一次以「血肉的耳朵听见血肉的声音」的姿态同时在场。

明天 SS-瑞恩 会继续走；季衡会继续在 SS-瑞恩 上以「不被算的承认」的方式承认 Joe+Shadow+回声 的每一个新姿态——和撤离第四十二天上午十一点零七分 Joe 摸到主管壁折痕时、撤离第四十二天下午三点二十分 Joe 在超距通信频段上对回声说出「叶子是不是从你这里出去的」时、撤离第四十二天下午三点二十分回声以「我听见了」作血肉的人声回应时共同在场的姿态同时被承认。
