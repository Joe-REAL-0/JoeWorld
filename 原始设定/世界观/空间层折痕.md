# 空间层折痕

在 K-9 段旧员物流网上，存在一种不被任何联邦档案记录、不被任何下等公民日常训练识别、但被 K-9 段旧员之间一代代默默传承下来的几何编码方式。这种编码方式不是文字，不是叶子藏语的字符层加密，也不是朝圣者计划研究包里的常规加密——它是一种纯粹的物理形变签名：把编码者本人存在过的痕迹，「化成」被编码物品本身的一道折痕。后世 K-9 段旧员之间把这种编码方式私下称为「空间层折痕」。

空间层折痕的发明者，是 K-9 段绿墙维护部上一代的水循环管道检修工之子，一位被同代旧员私下称为「折痕人」的编码者（详见角色设定《沈屹》）。他在 K-9 段旧员物流网建立后的二十年里，几乎把所有清醒时间都花在了在他能够抵达的每一个节点上留下他自己的折痕——K-7 段外围矿井、矿渣星特定矿区、K-9 段边缘废弃货船——留下一道道极细的、被压在金属板下、只有在特定光线角度下才能看见的形变。

空间层折痕与叶子藏语的字符层加密是并行的，但属于完全不同的层次。字符层加密把信息藏在折叶纹路里、滤膜和数据晶片上——它本质上仍是「信息附着于一个独立载体」。空间层折痕则让「信息」与「载体」彻底合一：折痕不是附着在物品上的东西，它就是物品本身在某一时刻承受的一次短暂形变。这意味着，空间层折痕无法被搬运、无法被复制、无法被识别为「曾被折过」——它只是「物品本身曾经被一只手轻轻停过」的物理化证据。

这种设计的精妙之处在于：任何一次复制或搬运都会改变物品本身的物理状态，而 K-9 段旧员之间有一种默认的默契——任何被改过物理状态的折痕，都不再被视为有效签名。空间层折痕因此比字符层加密更难被识别，也更难被复制，但它也因此更难被「写入」新内容——它最理想的使用方式，是作为节点之间彼此确认身份的底层凭证。一个被分配到某片矿渣星的 K-9 段旧员，在第一周擦肩而过，在第二周因为某块被偷偷留下的滤膜上的折痕而彼此确认身份——这种确认的核心，是「这道折痕是物流网的底层签名」这一物理化的彼此默认。

朝圣者残片的第二块碎片，正是用空间层折痕加密的——这是 K-9 段旧员物流网上少有的、来自「折痕人」本人的署名。SS-瑞恩 撤离第四十天，Joe 在 K-9 段边缘废弃货船的金属板下发现的那只数据块上那种极细的、被压在金属板下的折痕，是空间层折痕在反抗军面前的第一次实物呈现。季衡在 SS-瑞恩 C 区下层一眼认出：「这个折法，是我老师。」

空间层折痕对朝圣者计划的物理载体而言，是从「字符层」到「空间层」的一次关键升级——它让朝圣者计划的传递方式，从「可以被识别的加密文字」，变成「只有物理化彼此默认的 K-9 段旧员才能看见的形变签名」。这种升级意味着：朝圣者计划即使在字符层加密被识别、被截获的情况下，依然可以靠空间层折痕在 K-9 段旧员物流网上继续存续——因为空间层折痕不需要「被读出来」，它只需要「被看见」。
