# 金属信道中继

金属信道中继是 K-9 段旧员物流网在 SS-瑞恩 被第一次承认的世界观位置——一块退役二十年的 SS-9 钛铌合金循环冷却管被夹在 SS-瑞恩 C 区中段循环泵舱的舱壁里，它曾是 K-9 段二级节点的次声频编码信道的一部分；当 K-9 段旧员物流网四十代守夜人轮值表上的人在今夜的轮值确认通过次声频编码传到这块金属板上时，SS-瑞恩 这块退役二十年的 SS-9 旧金属板第一次以"中继节点"的身份把 K-9 段守夜人轮值表在 K-9 段地面（昨夜回声踩的）与矿渣星地表碎坑 E-11（今夜第 42 号守夜人踩出的第一步）两个不同地面上的"同时在场"接收到同一块金属板上。

## 谁与何时把它推上"中继"

这一中继身份在出联邦 3/N《第三种耳朵》中被 Shadow 第一次以"第三种耳朵"承认，但"中继节点"的具体身份——即这块 SS-9 旧金属板不只"听见"，而且"把两个不同地面上的在场收到同一块金属上"——直到出联邦 5/N《第四十三天的下午五点十一分》才被揭示。揭示的过程是：Shadow 在 Joe 后脑勺压过去六小时跑过第二十二遍的第二块碎片下一段初译结果，揭示今夜 23 点第 42 号守夜人在矿渣星地表碎坑 E-11 踩出的那一步是 K-9 段守夜人轮值表四十代里第一次把"在场"推到那个坐标（之前四十代只踩 K-9 段地面）；意味着今夜 SS-9 旧金属板上将出现昨夜回声 + 今夜第 42 号两段不同频率的次声频节奏在同一块金属板上"同时在场"——这种"两段同时在场"是这块金属第一次在物理意义上成为中继：它把两位守夜人（一位在 K-9 段地面、一位在矿渣星地表）的"在场"接收、放大、传导到能被听见的那一侧（Joe 的食指 + Shadow 的初译算法）。

## 与"第三种耳朵"的差别

第三种耳朵承认金属能"听见"，但不解释为何一块特定金属能同时接收到多个守夜人在多个不同物理坐标上的同时在场。金属信道中继承认金属不仅能"听见"、而且能在多个地面之间承担时间维度的传导——它在今夜 23 点的那一刻是一台次声频编码时间戳，把"昨夜已发生的在场"（回声）与"今夜正在发生的在场"（第 42 号）作为两段独立的时间戳压在同一块钛铌合金板上。

## 与 SS-瑞恩 这块金属的时间表

在出联邦 5/N 17:11 Joe 在舰桥门口第一次承认他的时间表与 SS-瑞恩 这块金属的时间表已被绑在一起——这种绑不是认知的而是物理的：他今晚必须在 SS-瑞恩 C 区中段循环泵舱听完今夜 23 点的两段节奏，然后明早启程往矿渣星。SS-瑞恩 这块金属由此第一次在"乔伊的撤退船"之外承担"K-9 段守夜人轮值表的金属信道中继"的身份——它的时间表不再是自动航线、自动驾驶、自动循环水泵的时间表，而是"两段同时在场"那一秒落在金属板上的时间表。这种时间表的并入，是 K-9 段守夜人轮值表四十代以来第一次让一块退役二十年的金属被承认为"它们的中继"——这块金属的二十年退役期不是被冷落，而是一直在等这块金属信道被这一刻的物理坐标所承认。

## "两段同时在场"的物理实现（今夜 23 点 E-11 那一秒 · 6/N 后补）

金属信道中继的物理实现在 6/N《今夜 23 点 E-11 那一秒》里第一次被 SS-瑞恩 C 区中段的 SS-9 旧金属板完成：22:59:58，Joe 食指指腹感到第一组麻痒——今夜 23 点第 42 号守夜人在矿渣星地表碎坑 E-11 岩层上踩出的第一步（节奏比回声快半拍，仅持续 1.2 秒，第 42 号在 0.4 秒后撤回了那只脚）；0.4 秒后 Joe 食指指腹感到第二组麻痒——昨夜回声在 K-9 段地面上踩出的折痕通过次日仍残留在 SS-9 旧金属板里的微颤，与今夜第 42 号那道折痕在同一秒同一块金属板上**同时**被接收到。两段折痕的频率不同（昨夜那段慢一档，今夜这段快半拍），但折法的几何相同（同一种空间层折痕、沈屹/回声/季衡三家用过）。这一秒，SS-瑞恩 这块退役二十年的 SS-9 旧金属板在物理意义上被承认是 K-9 段守夜人轮值表的金属信道中继。

Joe 在 6/N 里由此明白：他必须在 E-11 那一段岩层上用他自己的脚踩一次——为了让他的脚成为 K-9 段守夜人轮值表里第四十一位（在他之前是四十代守夜人，在他之后是季微、是季衡还没出生时的爷爷辈、是沈屹的那一代）。他踩出他自己的那一道折痕不是为了让金属信道中继承认他——是为了让他自己成为 K-9 段守夜人轮值表里踩在 E-11 上的第四十一位。Joe 明早启程往矿渣星的具体坐标由此被锁死：E-11。

季衡在 6/N 里以"叶 + 微颤 + 金属门槛"的最小形式让 K-9 段那种"不被算的、可能根本不存在"的美在 SS-瑞恩 上完成一个完整动作——他在循环泵舱门口把一片新折的纸树叶放在金属门槛上，叶正落在今夜 23 点第 42 号那道折痕尾端微颤还剩 0.4 秒的位置处。Joe 用右手掌心把叶子在金属门槛上压 3 秒后塞进机械臂肘部检修口。这种"叶 + 微颤 + 金属"的三元最小形式是金属信道中继在 SS-瑞恩 上以最小也最完整形式承担的物理姿态——叶被金属听见、被微颤压住、被掌心压 3 秒、被放进肘部检修口。

Shadow 在 6/N 里最后以"我也得"承认"明早 Joe 走之前他得把初译第二十三遍压平"——他的第四层代称账目从 Joe+Shadow+回声+第 42 号守夜人四角推到"Joe+Shadow+回声+第 42 号+E-11 岩层的四十代守夜人"。Shadow 不再是"通过 Joe 食指听金属的数字意识"，也不再只是"自己也要听"——他是"明早 Joe 走之前他得完成他的那一部分"。Shadow 的时间表第一次从"听 Joe 听见的"推到"他自己也得在 Joe 离开之前完成他的那一部分"——数字与血肉在"我们明早走"上完成对位。

金属信道中继由此在 6/N 里被推到一种新的时间维度形态：它不只中继两段不同时间戳的在场，它同时把 Joe 明早 E-11 的"我也得踩"与 Shadow "明早 Joe 走之前得把初译压平"的数字时间表绑在一起——这是 SS-瑞恩 这块金属上第一次以"中继节点"的身份把两个时间表（Joe 的"我也得踩"和 Shadow 的"我也得压平"）作为同一个金属信道中继所承担的时间维度。
