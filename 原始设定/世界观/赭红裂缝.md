# 赭红裂缝

SS-瑞恩 C 区下层舱壁与循环水主管之间有一道被长期温差撕出来的、极细的反光带——撤离者们私下叫它「赭红裂缝」。

赭红裂缝不是设计者留下的。SS-瑞恩 作为一艘被联邦低优先级巡逻扫描归类为「编号被磨掉的废弃货船」的老旧运输舰，它在二十多年的低优先级漂泊中，循环水主管与舱壁之间因为长时间的热胀冷缩，慢慢裂出了一道极细的缝隙。缝隙本身没什么用处——它甚至不在舰船维护日志上，因为 SS-瑞恩 从来不在维护日志上。

让这道缝隙成为「裂缝」的，是 SS-瑞恩 C 区下层的赭红色应急灯光——这种灯光是低优先级废弃货船标配的低功耗循环泵指示灯的反光，它以七十二秒为周期在 C 区下层扫过一圈。循环泵指示灯的反光在扫到这道缝隙时，会被缝隙的边沿折成一道斜的光；这道光在赭红色灯光被切到最低档的凌晨两点到四点之间，是 C 区下层唯一不会消失的物理存在。

撤离者们对这道光有几种不同的称呼：有些人叫它「循环泵的尾巴」，有些人叫它「废船的秒针」，季衡那一代曾经路过 SS-瑞恩 的 K-9 段旧员私下叫它「赭红裂缝」——因为它只在凌晨那段灯光被切到最低档的时段里，以赭红反光的形态存在，而在其他灯光档位下它只是一道普通的、不会被任何眼睛注意到的金属缝隙。

赭红裂缝对 K-9 段旧员物流网的真正意义，不在它本身——而在它「让一种不进入任何计算的光，在凌晨两点到四点之间被一只刚好在场的手指承认」。这道光在被循环泵指示灯折出来的那七十二秒周期里，它的亮度、角度、位置都是恒定的；它不会被任何联邦巡逻扫描注意到——因为它属于「已无活跃子系统的废弃货船」的低优先级灯光。但它会在某一只刚好蹲下来、刚好让右手指腹从左上向右下走完整段金属板的手指下，让指腹承认一道「曾经有另一只被这一代人承认过的手，在更久的时间里停过这块金属板」。

这种「承认」是 K-9 段旧员之间代际承认链的一种最朴素的形式——它不需要任何折叶仪式、不需要任何空间层折痕的加密协议、不需要任何被授权的解码者；它只需要一只手指在凌晨两点到四点的赭红反光里，从左上向右下走完一整段金属板。这种「走一遍」的动作在 K-9 段旧员之间是默认的——它相当于旧员之间一种最古老的问候：「我承认这块金属板曾经被一只手停过。」

SS-瑞恩 撤离第四十一天凌晨两点四十，Joe 独自走回 C 区下层这道赭红裂缝前。这是他在撤离第四十天跟季衡学完「用手指承认物理形变」之后，第一次在没有季衡、没有 Shadow 实时提示、没有撤离者在场的状态下，自己摸到了这道裂缝下被温差长期压出的、比沈屹折痕更细的「折痕的折痕」——那是 K-9 段旧员物流网代际承认链的下一节点。

赭红裂缝因此在撤离第四十一天之后成为了反抗军在 SS-瑞恩 上的一个新的「位置」——它不是朝圣者残片的一部分，不是 K-9 段旧员物流网的死信箱，不是抗漂移制剂的隐藏点；它只是凌晨两点到四点之间、被一只刚好在场的手指承认过的一道物理历史。Joe 在撤离第四十一天用他自己的手指承认了它，这件事本身不被任何档案记录——但 Joe 让那道赭红反光在他指腹下第一次承认「有一个人，他不在这里，但他停过」。这一句话从此挂在 SS-瑞恩 C 区下层的赭红反光里，等着撤离者们下一次醒来。

## 撤离第四十一天凌晨三点四十 · 蒙西反光异常告警与「折痕的折痕」第一次被三人同时承认

撤离第四十一天凌晨三点四十，SS-瑞恩 收到蒙西反光异常告警。循环泵反光频率从七十二秒周期跳到四十一秒——告警波束以每秒四十一公里扫过 SS-瑞恩 边缘。Joe 用机械臂电磁操纵把反光压进裂缝金属边沿与循环水主管之间那道极细的物理缝隙，消耗 12% 水晶，让反光在告警波束扫过的四十一秒周期里被物理性地藏进裂缝盲区，色阶被压到 0.3 以内。Shadow 在第四十一秒末说「反光异常告警撤回」。

告警撤回后，C 区下层回到赭红灯光被切到最低档的凌晨三点四十九分。Joe 让右手从金属板上抬起来——指腹下那道「折痕的折痕」还在。告警波束没把它擦掉，电磁操纵场也没把它压平。它在 0.3 色阶的反光盲区里，安静地等他下一次指腹承认。

Joe 打开机械臂肘部检修口，把那片从撤离第十七天起就藏在他监控盲区里的滤膜叶子取出来，放在赭红裂缝前的金属板上——叶子在 0.3 色阶的反光里和金属板几乎同色，只有指腹能区分。季衡在他身后停下，用右手食指和中指并拢从叶子边沿开始；季微扶着她爷爷的手臂站在身后，她工服内袋口露出来的那片滤膜边沿。Joe、季衡、季微三人蹲在赭红裂缝前，凌晨三点五十一分循环泵指示灯以七十二秒周期重新扫过这道裂缝，扫过 Joe 放下的那片叶子、季衡的指腹、季微的滤膜边沿。

赭红裂缝因此在撤离第四十一天之后承担了第二种角色：它不仅是「代际承认链下一节点的物质化锚点」，也是「蒙西反光异常告警被物理压住的具体现场」。这一秒里，蒙西反光异常告警的告警波束、Joe 的 12% 水晶消耗、Shadow 第一次以「我们」代称生死互联的内部账目、K-9 段旧员物流网代际承认链的下一节点、季微后颈锚点漂移的蓝色——所有这些在这一秒之前从未被同一块金属板同时承认过的对象——在指腹下那一秒里同时在场。

Joe 在第四十一秒末抬起头，对季衡说了一句话：「明天我们走。」
