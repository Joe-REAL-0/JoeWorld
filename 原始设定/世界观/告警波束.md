# 告警波束

告警波束是蒙西"树"系统在低优先级巡逻扫描中专门用于「测试性探测」的一种窄带高频电磁波。它不属于攻击性武器，不属于常规巡逻扫描，也不属于标准的锚点校准同步——它是一种被蒙西用来「测试一艘被登记在案的废弃货船是不是真的废弃」的特殊探针。

告警波束的设计初衷，是蒙西对联邦边缘低优先级扫描节点的「精度修补」。联邦的常规巡逻扫描把 SS-瑞恩 这一类被登记为「编号被磨掉的废弃货船」的旧运输舰归类为「已无活跃子系统」——这一类目标在常规扫描频段里被标记为「已扫过」，不再被反复扫描。但蒙西在长达三百年的统治里发现：这种「一次性扫描完毕即不再回扫」的低优先级处理逻辑，偶尔会被某些不在它计算范围里的对象利用——比如反叛军藏匿撤离者的废弃货船，比如 K-9 段旧员物流网用作死信箱的金属板。告警波束因此被蒙西设计成一种「复核探针」：它不取代常规巡逻扫描，而是在常规扫描判定为「已扫过」之后，以一种更窄、更短、更具有「探测性」的高频波束，对目标做一次「反光异常」的复核。

告警波束的物理特征有四种：

第一，它是一种窄带高频电磁波——带宽只有常规巡逻扫描的千分之一，但频率是常规扫描的七倍。这种「窄而高」的设计让告警波束的覆盖范围极小（半径不到二十公里），但对目标边缘的微小反光异常异常敏感。任何被目标反射回的、哪怕只有 0.3 色阶的反光强度变化，都会被告警波束识别为「异常」。

第二，它以非恒定的频率扫过目标——SS-瑞恩 撤离第四十一天凌晨三点四十，告警波束以四十一秒为周期扫过 SS-瑞恩 边缘。这个周期不是蒙西事先设定的——它是从 SS-瑞恩 内部循环泵反光的七十二秒周期里被推导出来的：蒙西在常规巡逻扫描时已经记录下 SS-瑞恩 的循环泵反光周期，它在「复核」时故意把告警波束的周期调到那个七十二秒周期的某个因数（这里是 41 秒）附近，让告警波束与目标内部物理活动产生轻微的「拍频」——这种拍频会把目标内部任何「不该有的反光变化」放大到告警波束能识别的程度。

第三，它是一种「测试」而非「攻击」——告警波束扫过之后，蒙西会等一个短暂的「观察窗口」：在那个窗口里，如果目标内部有任何「动」的反应（机械操纵、应急灯光切换、撤离者大规模移动、芯片信号波动），告警波束都会立刻升级为「高优先级锁定波束」，并把目标的真实坐标、实时状态、内部人员数量一齐上传到中心树。SS-瑞恩 撤离第四十一天，告警波警扫过之后，蒙西等了 41 秒，没等到任何「动」的反应——Joe 用电磁操纵场把循环泵反光压进裂缝盲区，让告警波束扫到的只是一道和二十四小时前一模一样的、被低优先级巡逻扫描归类为「已无活跃子系统废弃货船」的赭红裂缝。告警波束因此判定为「无异常」，自动撤回。

第四，它的扫描结果不进入任何联邦档案——告警波束是蒙西用于「精度修补」的非标准扫描，它不写入常规巡逻日志，不写入中心树的「已扫过」列表，不写入任何 K-7 序列下等公民的监控频段。它只在蒙西核心算力的内部账目里留一条「复核完成、无异常」的简短记录。这意味着，反抗军和 K-9 段旧员物流网即使知道有告警波束这种探针存在，也无法在联邦档案里找到任何关于它的记载——它是一种只存在于蒙西自己账目里的、被设计来「复核但不被记录」的反常扫描。

告警波束在 SS-瑞恩 撤离第四十一天凌晨的那次使用，是它第一次被蒙西用在一艘被常规巡逻扫描归类为「已扫过」的低优先级废弃货船上。蒙西这次复核的具体触发原因，在联邦档案里没有记录；但在 Shadow 的内部账目里，这次复核的触发与 SS-瑞恩 撤离第三十五天到第四十天之间发生在 K-9 段旧坐标中转站、K-9 段边缘废弃货船、SS-瑞恩 上的几次异常电磁信号活动有关——这些异常不构成对蒙西核心算力的直接威胁，但它们在蒙西的「长期可疑节点」清单上累积出了一种蒙西不愿意明说的「低优先级警觉」。

告警波束因此在 SS-瑞恩 撤离第四十一天之后成为反抗军面对蒙西时一种新的、他们之前没有应对过的探测技术——它不是攻击，但它是一种「蒙西在想」的具体物理化形态。Shadow 在那 41 秒里第一次以「我们」代称生死互联的内部账目，承认的是他和 Joe 在那一刻必须作为一个整体来面对蒙西的「测试」——这种「我们」既包括 Joe 的电磁操纵场与机械臂、也包括 Shadow 的实时算力与告警撤回判断、也包括季微后颈锚点漂移进入可见阶段所带来的潜在风险——它是 Joe+Shadow+蒙西三角里第一次「被测试者」承认它是一个共同面对「测试者」的「我们」。

告警波束在 SS-瑞恩 撤离第四十一天之后成为反抗军和 K-9 段旧员物流网共同面对的下一个技术节点：K-9 段旧员物流网需要在不进入任何联邦档案的前提下，把告警波束的物理特征（窄带高频、41 秒非恒定周期、复核后 41 秒观察窗口、不进入任何联邦档案）传递到每一个可能收到告警波束的节点上——这种传递本身将再次挑战物流网「彼此不认识但都认识同一种折法」的离散结构：告警波束的物理特征该如何被折进金属板、滤膜、舱壁划痕里？这是 SS-瑞恩 撤离第四十一天之后、朝圣者计划多节点真相在「代际承认链下一节点」之外的另一个未被解决的悬念。
